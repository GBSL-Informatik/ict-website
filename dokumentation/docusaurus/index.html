Was ist Sass?

Sass steht fÃ¼r Syntactically Awesome Style Sheets und ermÃ¶glicht das effizientere Schreiben von CSS-Code. Sass ist eine Erweiterung von CSS, welche insbesondere die Verwendung von verschachtelten Selektoren ermÃ¶glicht. Sass-Code wird in der Dateiendung .scss gespeichert. Sass-Code wird in CSS-Code kompiliert, welcher dann im Browser verwendet wird.

Statt CSS-Code wie folgt zu schreiben:

    background-color: #fff;
    color: #000;

}

    background-color: #fff;
    color: #000;

        background-color: #000;
        color: #fff;
    }
}
