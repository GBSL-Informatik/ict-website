# Bildschirmtimeout

:::tip[Anleitung]
Ausgehend von der Startseite :mdi[home]:

__Einstellungen :mdi[cog] > Präferenzen > Einstellungen Timer/Energie__ und dann auf 

__Zeit bis zur automatischen Rückstellung__
:::
